# GOMPact

**GOMPact** is a web-based platform designed to bridge the gap between Non-Governmental Organizations (NGOs) and volunteers during critical windows of disaster response.

This project was developed during **Bluehacks 2025**, a hackathon hosted by **ADMU's COMPSAT**, marking my first-ever hackathon experience.

The project was built under the themes of **SDG 11: Sustainable Cities and Communities** and **Disaster Risk Response**. In the Philippines, the speed of mobilization can save lives. GOMPact focuses on "compacting" the time it takes for NGOs to find verified, ready-to-help volunteers when a disaster strikes.
